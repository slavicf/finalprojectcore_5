
Экраны
Главный
    Отображение:    1. Текст: Приветственное сообщение
                    2. Текст: Название программы и ее версия
                    3. UI: Список действий на этом экране
    Действия:       1. Перейти в YouTube Analytics
                    2. Перейти в Настройки
YouTube Analytics
	Отображение:    1. Текст: YouTube Analytics
                    2. Текст: Выберите задачу
                    3. UI: Список действий на этом экране
	Действия:       1. Вернуться на главный экран
                    2. Перечисляются все задачи, выбрав одну из них - происходит переход на экран задачи

    Экран запроса
	Для некоторых запросов кол-во данных для отображения или для
	ввода может отличаться в зависимости от поставленных для
	запроса условий.
	Отображение:    1. Текст: Название запроса
                    2. Текст: Краткое описание результата выполнения запроса
                    3. Текст: Описание формата ввода данных
    Например: Введите [channelId_1] [channelId_2] через пробел и нажмите Enter
    Ввод:           1. Ввод данных в зависимости от запроса

Настройки
	Отображение:    1. Текст: Настройки
                    2. UI: Список действий на этом экране
	Действия:       1. Вернуться на главный экран
                    2. Изменить Использовать кэш
                    3. Изменить Путь к кэшу
                    4. Изменить Отображение времени затраченного не выполнение
    Использовать кэш
	Отображение:    1. Текст: Сохранять кэш?
                    2. Текст: True / False
    Ввод:           1. Текст: true / false (буквы могут быть большие либо маленькие, главное чтобы слово было верно написано)
    Путь к кэшу
	Отображение:    1. Текст: Путь к кэшу
                    2. Текст: [текущий путь]
    Ввод:           1. Текст: новый путь к кэшу
    Отображение времени затраченного не выполнение
	Отображение:    1. Текст: Отображать время затраченное на выполнение задачи?
                    2. Текст: True / False
    Ввод:           1. Текст: true / false (буквы могут быть большие либо маленькие, главное чтобы слово было верно написано)
